O primeiro passo é definir a estrutura narrativa. Com base no conceito "O Tempo das Coisas" desenvolvido no briefing, vamos criar um arco emocional que guie o espectador do início ao fim.

PROMPT 1: ESTRUTURA NARRATIVA 
Você é um roteirista especializado em vídeos publicitários e branded content. Preciso que crie a estrutura narrativa para um vídeo promocional de 60 segundos.

BRIEFING DO PROJETO:
- Marca: Raízes (café especial sustentável)
- Conceito central: "O Tempo das Coisas"
- Mensagem: Valorização do processo artesanal vs. instantaneidade moderna
- Tom: Cinematográfico, autêntico, emocional
- Público: Profissionais criativos 30-45 anos que valorizam qualidade

ESTRUTURA DESEJADA:
Divida o roteiro em 3 atos com timing específico:
- ATO 1 (0-15s): Gancho emocional - capturar atenção imediatamente
- ATO 2 (15-45s): Desenvolvimento - construir a narrativa da marca
- ATO 3 (45-60s): Resolução - call to action emocional

Para cada ato, defina:
1. Objetivo emocional (o que o espectador deve sentir)
2. Elemento visual principal (o que ele vê)
3. Progressão da narrativa (como avança a história)
4. Transição para o próximo ato

Mantenha coerência com o conceito "O Tempo das Coisas" - a ideia de que coisas boas levam tempo.

RESULTADO ESPERADO

ATO 1: O Convite (0-15s)

Objetivo: Criar curiosidade através do contraste

Visual: Mundo acelerado → transição para calma

Progressão: Do caos urbano ao primeiro grão de café

ATO 2: A Jornada (15-45s)

Objetivo: Construir conexão emocional com o processo

Visual: Mãos trabalhando, natureza, tempo passando

Progressão: Do plantio à xícara, mostrando cada etapa

ATO 3: O Momento (45-60s)

Objetivo: Criar desejo e pertencimento

Visual: Pessoa saboreando, sorriso sutil, logo

Progressão: Da preparação ao prazer do primeiro gole

2 Texto da Narração (Voiceover)

A narração é a alma do vídeo. Ela precisa ser poética mas acessível, emocional mas não piegas. Cada palavra conta em 60 segundos.

REGRAS PARA NARRAÇÃO DE 60s

  • • Palavras: Entre 90-120 palavras (ritmo natural de leitura)
  • • Pausas: Incluir momentos de silêncio para respiração visual
  • • Frases: Curtas e impactantes, fáceis de memorizar
  • • Última frase: Deve ser o slogan ou frase memorável da marca
PROMPT 2: TEXTO DO VOICEOVER 
Você é um copywriter especializado em branded content audiovisual. Crie o texto de narração (voiceover) para um vídeo de 60 segundos.

CONTEXTO DO PROJETO:
- Marca: Raízes - café especial de origem sustentável
- Conceito: "O Tempo das Coisas"
- Tom de voz: Calmo, reflexivo, poético mas acessível
- Narrador: Voz masculina madura, como quem conta uma história ao pé do fogo

ESTRUTURA NARRATIVA DEFINIDA:
- ATO 1 (0-15s): Contraste entre mundo acelerado e a proposta da marca
- ATO 2 (15-45s): Jornada do café - do cultivo à xícara
- ATO 3 (45-60s): O momento de saborear, conexão humana

REQUISITOS TÉCNICOS:
- Total: 90-120 palavras (para ritmo confortável)
- Inclua marcações de [PAUSA] onde o silêncio é intencional
- Inclua marcações de tempo aproximado [00:00]
- A última frase deve funcionar como slogan memorável
- Evite clichês como "o melhor café" ou "sabor incomparável"
- Use linguagem sensorial (texturas, aromas, temperaturas)

REFERÊNCIAS DE TOM:
- Documentários de natureza
- Propagandas da Apple (simplicidade poética)
- Narração de filmes do Terrence Malick

Crie o texto completo com todas as marcações.

3 Descrições de Cena Detalhadas

Agora vamos detalhar cada cena visualmente. Essas descrições servirão de base para gerar as imagens com IA e para criar o storyboard no próximo módulo.

PROMPT 3: DESCRIÇÕES DE CENA 
Você é um diretor de fotografia criando o shot list detalhado para um vídeo publicitário. Preciso de descrições visuais precisas para cada cena.

NARRAÇÃO DO VÍDEO:
[00:00-15:00 - ATO 1]
"Vivemos com pressa. Mensagens que não terminamos de ler. Cafés que não terminamos de sentir." [PAUSA]

[00:15-45:00 - ATO 2]
"Mas nas montanhas de Minas, o tempo ainda tem outro ritmo. [PAUSA] Mãos que conhecem cada grão pelo nome. Sol que amadurece sem pressa. Chuva que chega na hora certa. [PAUSA] Da terra ao torredor. Do torredor às suas mãos."

[00:45-60:00 - ATO 3]
"Cada xícara carrega essa jornada. Cada gole, uma pausa que você merece. [PAUSA] Raízes. O tempo das coisas boas."

DIREÇÃO VISUAL DEFINIDA:
- Paleta: Âmbar/marrom, verde esmeralda, creme quente, cinza pedra
- Estilo: Cinematográfico, luz natural, golden hour
- Referências: Documentários de natureza, comerciais premium
- Aspecto: 16:9 cinemático

PARA CADA CENA, FORNEÇA:
1. TIMING: Momento exato no vídeo (ex: 00:00-00:03)
2. TEXTO: Qual parte da narração acontece
3. ENQUADRAMENTO: Tipo de plano (close, médio, aberto, etc.)
4. DESCRIÇÃO VISUAL: O que vemos em detalhes
5. MOVIMENTO: Câmera estática, pan, tilt, dolly, etc.
6. LUZ: Tipo de iluminação da cena
7. HUMOR/EMOÇÃO: Sensação que a cena transmite

Crie entre 10-12 cenas para cobrir os 60 segundos completos.

5 Variações e Versões Alternativas

É sempre útil ter versões alternativas para diferentes contextos ou para apresentar opções ao cliente. Vamos criar variações do roteiro.

PROMPT 5: VERSÕES ALTERNATIVAS 
Com base no roteiro principal do vídeo "O Tempo das Coisas" para Raízes Café, crie versões adaptadas para diferentes formatos e contextos.

ROTEIRO PRINCIPAL (60s):
[Cole aqui o roteiro completo já aprovado]

CRIE AS SEGUINTES VERSÕES:

1. VERSÃO 30 SEGUNDOS (Instagram Reels / TikTok)
   - Condensar para os momentos mais impactantes
   - Manter a mensagem central intacta
   - Adaptar para formato vertical 9:16
   - Ritmo mais dinâmico mas sem perder a essência

2. VERSÃO 15 SEGUNDOS (Stories / Pré-roll)
   - Apenas o gancho principal + call to action
   - Uma única mensagem memorável
   - Máximo impacto em mínimo tempo

3. VERSÃO 90 SEGUNDOS (YouTube / Site)
   - Expandir com mais detalhes do processo
   - Adicionar momento de "behind the scenes"
   - Incluir mais informações sobre sustentabilidade
   - Ritmo mais contemplativo

4. VERSÃO SEM NARRAÇÃO (Redes Sociais)
   - Adaptar para funcionar só com texto na tela
   - Definir as frases-chave que aparecem como texto
   - Timing para leitura confortável

Para cada versão, forneça:
- Texto de narração adaptado (ou texto na tela)
- Quais cenas manter/cortar/adaptar
- Observações sobre ritmo e edição

POR QUE CRIAR VERSÕES?

• Multiplataforma: Cada rede tem formato ideal diferente
• Testes A/B: Testar diferentes durações e abordagens
• Orçamento: Cliente pode escolher versões conforme budget
• Acessibilidade: Versão sem áudio para quem assiste no mudo
